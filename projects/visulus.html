VISULUS█

Visulus is a B2B SaaS platform for asset lifecycle management, designed to help enterprise customers track warranty, compliance, insurance, and risk across their asset portfolio. The system features a multi-tenant architecture with PostgreSQL row-level security ensuring complete data isolation between organisations.

The frontend provides a clean dashboard built with React and TanStack Table, offering grouped table views with filtering, sorting, and column customisation. Assets can be tracked through their full lifecycle — from procurement through maintenance, compliance checks, and eventual decommissioning — with full audit trail support.

The backend follows a thin-handler pattern with FastAPI, where SQLModel provides unified ORM models that serve as both database tables and API schemas. Authentication flows through Auth0 with JWT validation, and the entire application is designed around a zero-border-radius aesthetic using the OKLCH colour space.
