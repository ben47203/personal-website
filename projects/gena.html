GENA█

Gena is an AI-powered automated collections system built for Australian businesses. It uses natural-sounding voice calls and personalised emails to contact customers with overdue invoices, integrating directly with accounting platforms like Xero and MYOB. The system ensures full compliance with Australian regulations including the ACCC, ASIC, Privacy Act, and ACMA guidelines.

The voice engine uses Pipecat for real-time AI conversations, with a cross-regional architecture that processes voice in the US for low latency while keeping all customer data and call recordings in Australia for data sovereignty. Calls are recorded in stereo WAV format and archived to S3, with full metadata extraction for audit trails.

The platform features a React dashboard for managing campaigns, reviewing call transcripts, and monitoring collection performance. Multiple AI providers can be swapped via environment variables, supporting Deepgram, Groq, Cartesia, and AWS alternatives for speech-to-text, language models, and text-to-speech.
