TDASH█

Tdash is a terminal dashboard that runs inside tmux, providing live system monitoring without ever leaving the terminal. It displays real-time CPU, memory, and network utilisation using colour-coded block graphs with historical data, alongside ASCII-art visualisations of each tmux window's pane layout.

The small monitor models represent each tmux window. Uniquely, the geometry of the models such as vertical and horizontal partitions represent the layout of the panes within each window. Further, the models actually resize to reflect the relative widths and heights of the actual panes.

A unique feature is Claude Code session detection — the dashboard identifies which tmux panes are running Claude Code and shows their current state (idle, working, or waiting for approval). It also tracks API token usage per session and weekly totals, parsed from Claude's JSONL log files.

The entire dashboard is built in Bash with no external dependencies beyond tmux, jq, and bc. It is designed for a specific screen width of roughly 167 columns — optimised for iPad Sidecar — with three monitors per row, periodic Cloudflare speed tests, and custom window naming via keybindings.
