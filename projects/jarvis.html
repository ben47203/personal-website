JARVIS█

Jarvis is a personal context assistant that runs as an MCP (Model Context Protocol) server, giving Claude deep, searchable access to my personal data across multiple communication platforms. It ingests messages from WhatsApp, iMessage, email, and Facebook Messenger into a unified SQLite database with full-text search powered by FTS5.

The system maintains a people model linking identities across platforms — so the same person's WhatsApp messages, emails, and iMessages are all connected. It includes a task management system with domain-based organisation, wake-date triggering, and conversation linking, plus a draft queue for AI-generated proposals like message drafts, task suggestions, and calendar events.

Jarvis exposes 26 MCP tools across messages, tasks, drafts, and sessions, with a smart sync pipeline that performs incremental data ingestion, automatic people linking, and LLM-powered analysis. A React dashboard provides a web interface for reviewing drafts, managing tasks, and browsing conversation history.
